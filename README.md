# Digital MEAL, XLSForms & Data Management Portfolio

مستودع رقمي مخصص لاستعراض وتوثيق نماذج الاستبيانات المتقدمة المصممة باستخدام معايير **XLSForm** والمجهزة للرفع والتشغيل المباشر على منصات جمع البيانات الميدانية مثل **KoboToolbox** و **ODK**، بالإضافة إلى مشاريع تنظيف، هيكلة، وتحليل البيانات للمشاريع الإنسانية والتنموية.

---

## 🚀 المشاريع المرفوعة في المستودع (Featured Projects)

### 📊 1. استبيان تقييم أوضاع الأسر (Comprehensive Household Assessment Survey)
* **الوصف:** مشروع متكامل لتنظيف وهيكلة بيانات استبيان ميداني شامل لتقييم الأوضاع الاجتماعية والديموغرافية والاقتصادية للأسر.
* **المهارات المطبقة:** 
  * تنظيف وتصفية البيانات واستبعاد المدخلات الشاذة أو غير المنطقية (Outliers & Typos).
  * توحيد النصوص وترميز المدخلات بصرياً وبرمجياً لتسهيل عمليات التحليل لاحقاً.
  * التحقق من سلامة البيانات الرقمية والتواريخ منطقياً.
* **رابط المجلد:** [عرض ملفات المشروع وتفاصيله](./Comprehensive-Household-Assessment-Survey)

### ⚙️ 2. تصميم الاستمارات المتقدمة - المجموعات المتداخلة (Advanced XLSForm: Nested & Conditional Groups)
* **الوصف:** نموذج عملي متقدم لبناء وتصميم الاستمارات المعقدة هندسياً وبرمجياً لتسهيل عمل فرق جمع البيانات الميدانية.
* **المهارات المطبقة:**
  * بناء المجموعات المتداخلة (Nested Groups) داخل الاستمارة.
  * تطبيق الشروط المنطقية الديناميكية (Skip Logic / Relevancies) لإظهار الأسئلة بناءً على الإجابات السابقة لتفادي أخطاء الإدخال الميداني.
* **رابط المجلد:** [عرض ملفات المشروع وتفاصيله](./Advanced-XLSForm-Nested-Groups)

### 📈 3. بيانات التقييم البعدي للأمن الغذائي - الخرطوم (FSL Khartoum PDM Data)
* **الوصف:** ملف بيانات تنظيف وتحليل لمشروع المتابعة والتقييم البعدي (Post Distribution Monitoring - PDM) الخاص بقطاع الأمن الغذائي وسبل العيش (FSL) في الخرطوم.
* **المهارات المطبقة:** تنسيق جداول البيانات الضخمة وتجهيزها لاستخراج المؤشرات الإحصائية المباشرة.
* **رابط الملف:** [عرض ملف البيانات المباشر](./FSL_Khartoum_PDM_Data.xlsx)
