نماذج الاستبيانات المتقدمة المخصصة لجمع البيانات الميدانية (XLSForms)

مستودع رقمي مخصص لاستعراض وتوثيق نماذج الاستبيانات المتقدمة المصممة باستخدام معايير *XLSForm* والمجهزة للرفع والتشغيل المباشر على منصات جمع البيانات الميدانية مثل *KoboToolbox* و *ODK*.

---

## 🛠️ المهارات والتقنيات التقنية المستخدمة:
* *تطوير النماذج الذكية:* بناء استمارات جمع بيانات متكاملة مبنية على منطق برمي متقدم داخل بيئة Microsoft Excel.
* *إدارة الحقول المتكررة (Repeat Groups):* استخدام المجموعات التكرارية لتسهيل إدخال بيانات أفراد الأسر أو الأنشطة الميدانية المتعددة بكفاءة عالية.
* *التحقق من جودة وصحة البيانات:* بناء قيود وشروط منطقية صارمة باستخدام الجمل الشرطية والـ (Constraints / Validation) لمنع إدخال البيانات الخاطئة أثناء العمل الميداني.
* *تنظيف البيانات وتكاملها:* استخدام دوال الإكسل المتقدمة (مثل VLOOKUP و TRIM) لتنقية البيانات قبل دمجها وتجهيزها للتحليل.

---
